Configuring Impersonation

In order to perform actions that require access to systems across a local network, you need to configure impersonation in the ASI client. Functions that require impersonation include:

  • Execution of programs that reside on network shared resources

  • Access to files residing on network shared resources

  • Automated deployment of the ASI client on systems in a local network

You will find impersonation configuration parameters in Scrip 43 (HandsFree Client Tools). The following sections include instructions for configuring impersonation at one site and on one system.

Site

  1. Log onto the ASI server (https://asiservername/main ). The ASI dashboard will be displayed.

  2. Click on the Use extended interface link on the right side of the ASI Dashboard upper pane, below user login and date. The Welcome page will be displayed.

  3. At the upper right-hand corner of any page on the ASI server extended interface, click on the configuration link in the sites navigation bar (https://asiservername/main/config/index.php?act=site ). The Select a Site page will be displayed.

  4. Locate the entry for the site where you want to configure Scrip 43, and click on its [configure site] link. The Scrip Configuration - Select Scrip - Site <site name> page will be displayed.

  5. Scroll down the page, locate the entry for Scrip 43 (HandsFree Client Tools), and click on its name. The Scrip 43 configuration page will be displayed.

  6. Scroll about three quarters down the Scrip 43 configuration page until you see the “Impersonation username” configuration parameter. Here, enter the name of the user the ASI client can impersonate as to access resources across the network. Leave the “Group” pull-down list to the left of the the “Impersonation username” configuration parameter label set to “site <”site name”>”.

  7. In the “Impersonation password” configuration parameter, found immediately below the “Impersonation username” configuration parameter, enter (twice) the password for the user name you entered in the “Impersonation username” configuration parameter. Leave the “Group” pull-down list to the left of the the “Impersonation username” configuration parameter label set to “site <”site name”>”.

  8. If the site you are configuring impersonation at is a Microsoft Windows Domain, in the “Impersonation domain” configuration parameter, enter the domain name for the site. Please note that you may see the domain for a site referred to as, for example, xxxxxx.domain.yyyyyy. In cases such as these, in the “Impersonation domain” configuration parameter, enter “domain”. Leave the “Group” pull-down list to the left of the the “Impersonation username” configuration parameter label set to “site <”site name”>”.

  9. Click on the “Submit Changes” button found both at the top and at the bottom of thhe Scrip 43 configuration page.

System

  1. Log onto the ASI server (https://asiservername/main ). The ASI dashboard will be displayed.

  2. Click on the Use extended interface link on the right side of the ASI Dashboard upper pane, below user login and date. The Welcome page will be displayed.

  3. At the upper right-hand corner of any page on the ASI server extended interface, click on the configuration link in the sites navigation bar (https://asiservername/main/config/index.php?act=site ). The Select a Site page will be displayed.

  4. Locate the entry for the site where the system whose Scrip 43 you want to configure is located, and click on its [view machines] link. The Configuration - <site name> Machines page will be displayed.

  5. Locate the entry for the system whose Scrip 43 you want to configure, and click on its [configure Scrips] link. The Scrip Configurator - Select Scrip - Site <site name> - Machine <machine name> page will be displayed.

  6. Scroll down the page, locate the entry for Scrip 43 (HandsFree Client Tools), and click on its name. The Scrip 43 configuration page will be displayed.

  7. Follow steps 6. to 9. in the “Site” section of this document, with one change. In the “Group” pull-down list for each of the Scrip 43 configuration parameters you are configuring, select “machine <”machine name”>”.
