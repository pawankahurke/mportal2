How to customize and enable an event notification

  1. Log onto your ASI server (https://asiservername/main). The ASI dashboard will be displayed.

  2. Click on the "Use extended interface" link on the right side of the ASI Dashboard upper pane, below user login and date. The Welcome page will be displayed.

  3. At the upper right-hand corner of any page on the ASI server, click on the “notifications” link in the “events” navigation bar found the at upper right-hand corner of any page in the ASI server extended interface. The “Event Notifications” page will be displayed.

  4. To retrieve the notifications you want to customize and enable, in the "Name Contains" parameter in the "Search Options" panel, enter one or more keywords the name of these notifications have in common. If necessary, adjust the value of the “Page Size” parameter in the “Display Options” panel.

  5. Click on the Search button. The notifications you want to customize and enable will be listed in the “Event Notifications” page.

  6. If all you want to do is enable an event notification, just click on its “enable” action link, found in the “Action” column to the left of the name of the notification.

  7. If you want to customize the event notification, then click on the “edit” action link , found in the “Action” column to the left of the name of the notification. The “Edit an Event Notification” page will be displayed (the name of the page will be slightly different if you are customizing a global event notification owned by another user).

  8. Review the configuration settings of the event notification you want to customize, and change them as needed carefully following the on-screen instructions. In order to enable the event notification select “Yes” from the “Enabled” configuration parameter pull-down list. Once you have completed the review and change of the event notification configuration, click on the “Update” button (if you are customizing a global event notification owned by another user, you will click on the “Add” button as you will be adding a local copy of the event notification which you will own).
