How to Add a User átop

1) Log onto the ASI server (https://asiservername/main)

2) Click on the "Use extended interface" link on the right side of the ASI Dashboard upper pane, below user login and date.

3) Go to the admin page ("tools" navigation bar >>> admin (https://asiservername/main/acct/admin.php))

4) Click on the "new user" link in the list of links above the user table. The "Create User" page will be displayed (https://asiservername/main/acct/adm-act.php?action=ccu)

5) Enter the information for the new user following the instructions in the attached document (HFN asi server user - site setup - 08.doc)

6) Click on the "create" button at the bottom of the page

Based on past experience, we would suggest that you give the new user "Normal" privileges, and additional rights only needed in order to perform job related functions (e.g. if you want the user to configure Scrips, you would give him/her "Configure machines" rights.)

How to access and change the default event notification and report recipient átop

You can access and edit the default event notification and report e-mail recipient(s) defined on your ASI server as follows:

1) Log onto the ASI server (https://asiservername/main)

2) Click on the "Use extended interface" link on the right side of the ASI Dashboard upper pane, below user login and date.

3) Go to the "Admin" page (click on the admin link in the tools navigation bar at the top right-hand corner of any page on the ASI server extended interface (https://asiservername/main/acct/admin.php),)

4) Scroll down to the bottom of the "Admin" page. There, you will see the default notification and report e-mail addresses

5) You can change the content of the "Notifications default email recipients" and "Report default email recipients" configuration parameters. If you do, click on the "Update" button for the changes to take effect.

Please note that the master user on your ASI server can change the default event notification and report recipients for any user account on the ASI server. In order to do that:

1) Log onto the ASI server (https://asiservername/main) as the master user

2) Click on the "Use extended interface" link on the right side of the ASI Dashboard upper pane, below user login and date.

3) Go to the "Admin" page (click on the admin link in the tools navigation bar at the top right-hand corner of any page on the ASI server extended interface (https://asiservername/main/acct/admin.php),)

4) Scroll down to the entry for the user whose default event notification or report recipient you want to access or modify, and click on the “edit” link to the left of the user name. The “Edit User” page will be displayed

5) Modify the entries in the “Notify mail” and “Report mail” fields as desired.

6) Click on the “Update” button.

How to access or modify the e-mail address for a site átop

1) Log onto the ASI server (https://asiservername/main) as the master user

2) Click on the "Use extended interface" link on the right side of the ASI Dashboard upper pane, below user login and date.

3) Go to the "Admin" page (click on the admin link in the tools navigation bar at the top right-hand corner of any page on the ASI server extended interface (https://asiservername/main/acct/admin.php),)

4) Scroll down to the entry for the site whose e-mail address you want to access or modify, and click on the “edit” link to the left of the site name. The “Set Site Owner” page will be displayed.

5) Enter the desired e-mail address, or modify the existing one in the “Site Email Address” field

6) Click on the “Assign” button
