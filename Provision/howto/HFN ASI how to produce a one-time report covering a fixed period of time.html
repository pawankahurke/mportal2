How to Produce a One-time Report Covering a Fixed Period of Time



  1. Log onto your ASI server (https://asiservername/main). The ASI dashboard will be displayed.

  2. Click on the "Use extended interface" link on the right side of the ASI Dashboard upper pane, below user login and date. The “Welcome” page will be displayed.

  3. Click on the “reports” link in the “reports” navigation bar found the at upper right-hand corner of any page in the ASI server extended interface. The “Reports” page will be displayed.

  4. In order to access the report you want to produce for a fixed period of time,

      • Enter one or more character strings from the name of the report you want to copy or edit in the "Name Contains" parameter in the "Search Options" panel on the “Reports” page.

      • If necessary, adjust the value of the “Page Size” parameter in the “Display Options” panel, then

      • Click on the “Search” button.

      • Locate the entry for the report you want produce, and click on its “copy” link (found in the “Action” column to the left of the report name). The “Copy a Report” page will be displayed.

  1. Change the report name as desired

  2. If you want to produce a report covering all sites, make sure the “Global” configuration option is not selected. If you want to produce a copy of the report for each user with an account on the ASI server (covering the sites each user has access to), select the “Global” configuration option.

  3. If you want each user on the server to have their own report but not the user you logged on as, de-select the “Do not generate report for owner” configuration option.

  4. Make sure the state of the report is enabled. Select “Enabled from the “State” pull-down list.

  5. Set the report's one time schedule as follows:

  • In the “Schedules” section of the “Copy a Report” page, click on the “Run Once” tab

  • In the “Enter the date and time for this one-time event:” enter a time in the near future when you want the report to run (to be safe, select a time at least 10 minutes in the future).

  • In the “Start time” field enter the starting date and time of the report coverage period.

  • In the “End time” field enter the ending date and time of the report coverage period.

  • For example,

  • Start time – 12/01/07 00:00

  • End time – 01/01/08 00:00

  1. Click on the “Add” button.

Please note that you can also access the “reports” module on the ASI server also via the “reports” link found in the row of links at the top and bottom of any of the “Sections” pages (https://asiservername/main/report/report.php?section=0&act=0).
