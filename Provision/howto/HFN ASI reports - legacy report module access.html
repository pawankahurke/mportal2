How to Access the Automated Support Infrastructure (ASI) Legacy Report Modules

Although the preferred ASI reporting function is the one you access directly via the “reports” navigation bar found on any page in the ASI server extended interface, you can still access the older reporting modules where all the reports you have been using are still available. To do this, please perform the following steps:

  1. Log onto your ASI server (https://asiservername/main). The ASI dashboard will be displayed.

  2. Click on the "Use extended interface" link on the right side of the ASI Dashboard upper pane, below user login and date. The Welcome page will be displayed.

  3. Click on the “legacy reports” link in the “reports” navigation bar found the at upper right-hand corner of any page in the ASI server extended interface. The “Legacy Reports” page will be displayed.

  4. Click on the link corresponding to the legacy report module you want to access found in the body of the “Reports – Legacy” page:

    • “Legacy event reports”. Click on this link to access the “legacy” event reports page at https://asiservername/main/event/report.php

    • “Legacy asset reports”. Click on this link to access the “legacy” asset reports page at https://asiservername/main/asset/report.php

  1. Depending on the link you clicked on, the “legacy” “Event Reports” and “Asset Reports” pages will be displayed.
