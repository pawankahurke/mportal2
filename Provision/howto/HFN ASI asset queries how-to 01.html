How to export the list of currently defined asset queries átop

1) Log onto your ASI server (https://asiservername/main)

2) Click on the "Use extended interface" link on the right side of the ASI Dashboard upper pane, below user login and date.

3) In the "sites" navigation bar click on the "wizard" link (https://asiservername/main/config/index.php?act=wiz)

4) In the "Configuration Wizard" page, click on the "Export data" entry (https://asiservername/main/acct/export.php)

5) Click on the "asset" radio button, then click on the "Next" button

6) Click on the "AssetSearches" radio button, then click on the "Next" button

7) De-select the following fields in the AssetSearches table:

  • id

  • username

  • rowsize

  • modified

  • global

  • refresh

  • translated

  • date_code

  • expires

  • date_value

  • created

8) Select 0 from the "Select a maximum number of records to export:" pull-down list

9) Select the desired date format from the "Format time" pull-down list

10) Click on the "Next" button

11) Enter the desired file name in the "Filename" data entry box

12) Click on the "Text" button

13) Save the file to the desired location.

How to duplicate an asset query átop

If you want to create your own variant of an asset query, for example, the "Hardware IV" asset query, you would:

1) Log onto your ASI server, https://asiservername/main/index.php

2) Click on the "Use extended interface" link on the right side of the ASI Dashboard upper pane, below user login and date.

3) In the "assets" navigation bar (at the upper right-hand corner of any ASI server page), click on the "queries" link

4) In the "Name Contains" parameter in the "Search Options" panel, enter "hardware" (without quotes)

5) Click on the "Search" button

6) Click on the "copy" link to the right of the asset query name. In this case, you would click on the "copy" link for the "Hardware IV" query. Please note that depending on the CPU power of the system you are using it may take 3-30 seconds for the "Duplicate Asset Query" page to be fully displayed. If a warning dialog box asking you "Do you want to abort the script" appears, click on the "No" button.

7) Change the name of the asset query

8) De-select the "Global" configuration parameter, or leave it unchanged

9) Modify the selection in the "Select fields to display" panel on the left side of the page

10) If you want to add or modify search criteria, make the changes in the "Create search criteria" panel on the right side of the page. For example, if you wanted to base the search criteria on the site name, you would click on the "Site Name" field label in the "Properties" asset item group in the "Select fields to display" panel on the left side of the page. If you wanted to add more search criteria, you would click on the "AND", or "OR" buttons, and select another criteria.

11) Once you have completed steps 8) and 9), you should review the configuration settings in the "Select date" and "Select display options" sections of the "Duplicate Asset Query" page

12) Click on the "Update" button

How to export asset query results átop

If you want to export the asset information retrieved by an asset query please perform these steps:

1) Log onto your ASI server, https://asiservername/main/index.php

2) Click on the "Use extended interface" link on the right side of the ASI Dashboard upper pane, below user login and date.

3) In the "assets" navigation bar (at the upper right-hand corner of any ASI server page), click on the "queries" link

4) Retrieve the entry for the asset query whose results you want to export either by scrolling down to the entry, or by retrieving the entry for the asset query using the search parameters in the "Search Options Panel".

5) Click on the "export" link to the left of the name of the asset query whose results you want to export. The export control panel will be displayed in a new window

6) You have two options, export to a file in command delimited text file with headers (.csv), or to an SQL table. If you want to export to a .csv file, enter the name of the file name you want to export the asset information to, then click on the "Excel" button.

7) The "File Download" dialog box will be displayed. Click on the "Save" button

8) After you have browsed to the desired location, and made sure that the file name is as you want it, click on the "Save" button.
