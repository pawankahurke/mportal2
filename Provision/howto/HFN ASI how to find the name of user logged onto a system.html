How to Find the Name of the User Logged onto a System



  1. Log onto your ASI server (https://asiservername/main). The ASI dashboard will be displayed.

  2. Click on the "Use extended interface" link on the right side of the ASI Dashboard upper pane, below user login and date. The Welcome page will be displayed.

  3. In the "events" navigation bar found at the top right-hand corner of any page of the ASI server extended interface, click on the "ad-hoc query" link. The "Ad-hoc Event Query From" page will be displayed.

  4. In the "Select Additional Search Criteria for Events: " area, from the "Machine" pull-down list select the name of the system whose currently logged on user you want to find

  5. In the "Narrow Your Search (optional): ", From the "Scrip number: " pull-down list select 174

  6. Click on the "Search" button. The “Event Query Results” page will be displayed.

  7. If necessary, repeat the search increasing the period covered by changing the values of the "Date/Time from:" and "Date/Time to:" parameters.
