How to Perform Report Related Activities

  • Adding, a report

  • Copying, or editing a report

  • Running a report on-demand

  • Deleting a report



Adding a report átop

  1. Log onto your ASI server (https://asiservername/main). The ASI dashboard will be displayed.

  2. Click on the "Use extended interface" link on the right side of the ASI Dashboard upper pane, below user login and date. The “Welcome” page will be displayed.

  3. Click on the “reports” link in the “reports” navigation bar found the at upper right-hand corner of any page in the ASI server extended interface. The “Reports” page will be displayed.

  4. Click on the “add report” link found below the page title. The “Add a Report” page will be displayed.

  5. Follow the on-screen instructions carefully.

  1. After you finish defining a Report, click on the “Add” button found at the top and bottom of the page.

Please note that you can also access the “reports” module on the ASI server also via the “reports” link found in the row of links at the top and bottom of any of the “Sections” pages (https://asiservername/main/report/report.php?section=0&act=0).

Copying, or editing a report átop

In order to edit or copy a report, please perform the following steps:

  1. Log onto your ASI server (https://asiservername/main). The ASI dashboard will be displayed.

  2. Click on the "Use extended interface" link on the right side of the ASI Dashboard upper pane, below user login and date. The “Welcome” page will be displayed.

  3. Click on the “reports” link in the “reports” navigation bar found the at upper right-hand corner of any page in the ASI server extended interface. The “Reports” page will be displayed.

  4. In order to access the report you want to edit or copy:

      • Enter one or more character strings from the name of the report you want to copy or edit in the "Name Contains" parameter in the "Search Options" panel on the “Reports” page.

      • If necessary, adjust the value of the “Page Size” parameter in the “Display Options” panel, then

      • Click on the “Search” button.

      • Locate the entry for the report you want to edit or copy, and click on its “edit” or “copy” link (found in the “Action” column to the left of the report name). The “Edit a Report” or “Copy a Report” page will be displayed.

  1. Follow the on-screen instructions carefully.

  2. Click on the “Update” (if you are editing a report), or “Add” (if you are copying a report) buttons. Note that if you are editing a report you do not own, the button label in the “Edit a Report” page will be “Add” as you will be adding a copy of the report which you will own.

Running a report on demand átop

In order to run a report on demand, please perform the following steps:

  1. Log onto your ASI server (https://asiservername/main). The ASI dashboard will be displayed.

  2. Click on the "Use extended interface" link on the right side of the ASI Dashboard upper pane, below user login and date. The “Welcome” page will be displayed.

  3. Click on the “reports” link in the “reports” navigation bar found the at upper right-hand corner of any page in the ASI server extended interface. The main “Schedules” page will be displayed.

  4. In order to access the report you want to run on demand:

      • Enter one or more character strings from the name of the report you want to run on demand in the "Name Contains" parameter in the "Search Options" panel on the “Reports” page.

      • If necessary, adjust the value of the “Page Size” parameter in the “Display Options” panel, then

      • Click on the “Search” button.

  1. Click on the “run now” link found in the “Action” column for the report you want to run on demand.

      • If the report you want to run on demand has only one schedule, its output for the period covered by the assigned schedule will be displayed in a new window.

      • If more than one schedule is assigned to the report you want to run on demand, The “Report Run Now” will be displayed listing all of the schedules assigned to the report, and asking you to select the one you want to use for the on-demand execution of the report by clicking on its name. After you click on the name of the schedule you want to use for the on-demand execution of the report, ts output for the period covered by the selected schedule will be displayed in a new window.

In either of the above cases, the report will also be sent to the other output destinations it is configured for, e-mail recipients, and/or the information portal.

Deleting a report átop

In order to delete a report, please perform the following steps:

  1. Log onto your ASI server (https://asiservername/main). The ASI dashboard will be displayed.

  2. Click on the "Use extended interface" link on the right side of the ASI Dashboard upper pane, below user login and date. The “Welcome” page will be displayed.

  3. Click on the “reports” link in the “reports” navigation bar found the at upper right-hand corner of any page in the ASI server extended interface. The main “Schedules” page will be displayed.

  4. In order to access the report you want to delete:

      • Enter one or more character strings from the name of the report you want to delete in the "Name Contains" parameter in the "Search Options" panel on the “Reports” page.

      • If necessary, adjust the value of the “Page Size” parameter in the “Display Options” panel, then

      • Click on the “Search” button.

  1. Click on the “delete” link found in the “Action” column for the report you want to delete. The “Report <schedule name> Deletion” page will be displayed.

  2. Click on the “Yes” link. The “Report <schedule name> Deletion” confirmation page will be displayed.
