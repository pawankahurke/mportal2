Scrip 238 - Symantec Anti Virus Definition Dates Log

Scrip 238 reports the Symantec virus definition dates on the system where the Scrip is executed, and the latest virus definition dates on the sub-net where the system is located.

This Scrip can be executed on a schedule or on demand.

The table below describes the information contained in the logs produced by this Scrip.


Client time:

Date and time at which event was detected by the Client on the end-user system (e.g. 2001-03-02 19:21:36)

Server time:

Date and time at which event log was received and stored in the SQL database on the Web log server (e.g. 2001-03-02 19:21:53)

Scrip number:

Number of Scrip (no particular relevance)

Customer:

Name of supported customer, individual department or site

Machine:

Name of system as assigned at the time of Windows configuration

Username:

Name of user when event occurred

Client version:

Client version number (e.g. 0.700.2048.LE)

Client size:

Size of Client in bytes

Priority:

Priority of event ranging between 1 (lowest) and 9 (highest)

Description:

Description of event that took place (e.g. symptom detected, problem resolved or automated procedure executed)

Type:

n/a

Path:

n/a

Executable:

n/a

Version:

n/a

Size:

n/a

ID:

n/a

Window title:

n/a

String 1:

Date of virus definitions on the system Scrip 238 ran on. The date is stored in POSIX format.

String 2:

Date of latest virus definitions on the sub-net where the system whose Scrip 238 posted the event log is located on. The date is stored in POSIX format.

Text 1:

Virus definition dates on the system where Scrip 238 ran, and the latest virus definition date on the sub-net.

(E.g.

Symantec virus definition dates on this system: 16-Jun-2004, latest on network: 16-Jun-2004

)

Text 2:

Date of the virus definitions on the system Scrip 238 ran on

(E.g.

Wednesday, June 16, 2004 00:00:00

)

Text 3:

Date of latest virus definitions available on the sub-net

(E.g.

Wednesday, June 16, 2004 00:00:00

)

Text 4:

Scrip execution type and anti virus product name and version

(E.g.

Data: Type of run: Scheduled
Data: Product: Norton AntiVirus 4
Data: Version: 5.0.0.26

)
