Scrip 245 – On-demand GoToAssist

Scrip 245 automates the establishment of a GoToAssist remote control session adding a number of significant benefits to GotoAssist functions:

  • It lets users control computers remotely without end-user intervention. This makes it ideal for remote control of unattended systems, e.g. Servers.

  • It lets users take remote control of systems even when they are logged off.

  • Scrip 245 brings scalability to remote control. The GoToAssist configuration on one, some, or all systems on a sub-net is completely automated, and managed via a single Scrip configuration page.
