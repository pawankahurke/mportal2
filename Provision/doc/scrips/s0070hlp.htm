Scrip 70 - Software Installation Detected

Scrip 70 detects and reports software installations that take place on systems on which it is installed. The default values for the file masks used by Scrip 70 to detect installation of a software program are:

  • _ins????._mp
  • *setup*.exe
  • appinst.exe
  • asinstal.exe
  • inst*.exe
  • load.exe
  • run.exe
  • startup.exe
  • startwin.exe
  • tuavinst.exe
  • wininst.exe
  • winload.exe
  • winstart.exe
  • wintst.exe
  • wksbinst.exe
  • msiexec.exe
  • Uedit32i.exe

The file masks can be changed using the Scrip configurator.

Once triggered, Scrip 70 checks the Microsoft registry for changes before reporting a successful software installation.

The table below describes the information contained in the logs produced by this Scrip.


Client time: Date and time at which event was detected by the Client on the end-user system (e.g. 2001-03-02 19:21:36)
Server time: Date and time at which event log was received and stored in the SQL database on the Web log server (e.g. 2001-03-02 19:21:53)
Scrip number: Number of Scrip (no particular relevance)
Customer: Name of supported customer, individual department or site
Machine: Name of system as assigned at the time of Windows configuration
Username: Name of user when event occurred
Client version: Client version number (e.g. 0.700.2048.LE)
Client size: Size of Client in bytes
Priority: Priority of event ranging between 1 (lowest) and 9 (highest)
Description: Description of event that took place (e.g. symptom detected, problem resolved or automated procedure executed)
Type: n/a
Path: n/a
Executable: n/a
Version: Version of program used to install the software application

(e.g. 1.5 Build 14)

Size: Size of program used to install the software application in bytes

(e.g. 218624)

ID: n/a
Window title: n/a
String 1: Creation date of program used to install the software application

(e.g. Friday, June 5, 1998 04:01:08)

String 2: Client location on system that reported the event

(e.g. C:\PROGRAM FILES\...\CLIENT\CUST.EXE)

Text 1:
Text 2: Process id of program used to install the software application

(e.g. FFFB9DF7)

Text 3:
Text 4: Location of program used to install the software application and name of program that was installed

(e.g.
