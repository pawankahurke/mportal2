Scrip 71- Software Removal Detected

Scrip 71 detects and reports the removal of software applications that when installed were registered and appear in the window of the Install/Uninstall tab of the Add/Remove Program Properties panel. This Scrip is triggered when the Client detects the execution of software removal programs that match the following masks:

  • isuninst.exe
  • uninst*.exe
  • unwise*.exe
  • msiexec.exe

Once triggered, Scrip 71 checks the Microsoft registry for changes before reporting a successful software removal.

The table below describes the information contained in the logs produced by this Scrip.


Client time: Date and time at which event was detected by the Client on the end-user system (e.g. 2001-03-02 19:21:36)
Server time: Date and time at which event log was received and stored in the SQL database on the Web log server (e.g. 2001-03-02 19:21:53)
Scrip number: Number of Scrip (no particular relevance)
Customer: Name of supported customer, individual department or site
Machine: Name of system as assigned at the time of Windows configuration
Username: Name of user when event occurred
Client version: Client version number (e.g. 0.700.2048.LE)
Client size: Size of Client in bytes
Priority: Priority of event ranging between 1 (lowest) and 9 (highest)
Description: Description of event that took place (e.g. symptom detected, problem resolved or automated procedure executed)
Type: n/a
Path: n/a
Executable: n/a
Version: Version of program used to remove the software application

(e.g. 5, 51, 138, 0)

Size: Size of program used to remove the software application in bytes

(e.g. 306688)

ID: n/a
Window title: n/a
String 1: Creation date of program used to remove the software application

(e.g. Saturday, March 10, 2001 22:11:18)

String 2: Client location on system that reported the event

(e.g. C:\PROGRAM FILES\...\CLIENT\CUST.EXE)

Text 1: n/a
Text 2: Process id of program used to remove the software application

(e.g. FFFCA1DF)

Text 3: n/a
Text 4: Location of program used to remove the software application and name of program that was removed

(e.g.

Started uninstall: C:\WINDOWS\ISUNINST.EXE
The following program has been uninstalled: Heroes of Might and Magic® III
