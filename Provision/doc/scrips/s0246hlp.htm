Scrip 246 – Network Device Discovery

Scrip 246 reports the content of the ARP table listing all devices on the LAN the system where Scrip 246 runs is connected to.

Scrip 246 can run in one of three modes

  • On a user defined schedule

  • On demand, triggered by a user

  • Triggered when the ASI network driver dynamically detects a change in the ARP table

For each device on the LAN, Scrip 246 reports:

  • The NetBIOS name or fully qualified domain name

  • The IP address

  • The MAC address of the devices's network interface card

The table below describes the information contained in the logs produced by Scrip 246.

Client time:

Date and time at which event was detected by the Client on the end-user system (e.g. 2001-03-02 19:21:36)

Server time:

Date and time at which event log was received and stored in the SQL database on the Web log server (e.g. 2001-03-02 19:21:53)

Scrip number:

Number of Scrip (no particular relevance)

Customer:

Name of supported customer, individual department or site

Machine:

Name of system as assigned at the time of Windows configuration

Username:

Name of user when event occurred

Client version:

Client version number (e.g. 0.700.2048.LE)

Client size:

Size of Client in bytes

Priority:

Priority of event ranging between 1 (lowest) and 9 (highest)
