Scrip 15 - Anti Virus Scan Execution

Scrip 15 executes the Norton Anti Virus virus scan, and reports its completion.

The table below describes the information contained in the logs produced by this Scrip.


Client time:

Date and time at which event was detected by the Client on the end-user system (e.g. 2001-03-02 19:21:36)

Server time:

Date and time at which event log was received and stored in the SQL database on the Web log server (e.g. 2001-03-02 19:21:53)

Scrip number:

Number of Scrip (no particular relevance)

Customer:

Name of supported customer, individual department or site

Machine:

Name of system as assigned at the time of Windows configuration

Username:

Name of user when event occurred

Client version:

Client version number (e.g. 0.700.2048.LE)

Client size:

Size of Client in bytes

Priority:

Priority of event ranging between 1 (lowest) and 9 (highest)

Description:

Description of event that took place (e.g. symptom detected, problem resolved or automated procedure executed)

Type:

n/a

Path:

n/a

Executable:

n/a

Version:

n/a

Size:

n/a

ID:

Completion code corresponding to completion message reported in field string1.

0 "Scan completed successfully or was terminated by user."

1 "No scan targets were specified."

2 "Scan executable could not be determined."

3 "Scan executable could not be found."

4 "Command line for scan executable could not be determined."

5 "Report file path could not be determined."

6 "Failed to run the scan executable."

7 "Scan was initiated and is now running."

8 "Failed to created registry controlled scan."

9 "Failed to start registry controlled scan."

10 "Attempt to execute scan failed."

11 "Scheduler error encountered."

12 "Version no longer supported by vendor."

13 "Version supported by this Scrip not detected."

14 "Virus scanning status code not set."

Window title:

n/a

String 1:

Completion status

(e.g. Scan completed successfully or was terminated by user. )

String 2:

Client location on system that reported the event

(e.g. C:\PROGRAM FILES\...\CLIENT\CUST.EXE)

Text 1:

Scrip execution completion message

(e.g. Scanning viruses is done.)

Text 2:

Type of Scrip execution, product name and product version

(e.g.

Type of run: Scheduled
Product: Norton AntiVirus 4
Version: 5.0.0.26

)

Text 3:

Command line executed by Scrip and program execution log

(e.g.

)
