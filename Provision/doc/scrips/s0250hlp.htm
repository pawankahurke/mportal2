Scrip 250 - Trend Micro Anti Virus Definition Dates Log

Scrip 250 reports the Trend Micro Anti Virus virus definition dates on the system where the Scrip is executed, and the latest virus definition dates on the sub-net where the system is located.

This Scrip can be executed on a schedule or on demand.

The table below describes the information contained in the logs produced by this Scrip.


Client time:

Date and time at which event was detected by the Client on the end-user system (e.g. 2001-03-02 19:21:36)

Server time:

Date and time at which event log was received and stored in the SQL database on the Web log server (e.g. 2001-03-02 19:21:53)

Scrip number:

Number of Scrip (no particular relevance)

Customer:

Name of supported customer, individual department or site

Machine:

Name of system as assigned at the time of Windows configuration

Username:

Name of user when event occurred

Client version:

Client version number (e.g. 0.700.2048.LE)

Client size:

Size of Client in bytes

Priority:

Priority of event ranging between 1 (lowest) and 9 (highest)

Description:

Description of event that took place (e.g. symptom detected, problem resolved or automated procedure executed)

Type:

n/a

Path:

n/a

Executable:

n/a

Version:

n/a

Size:

n/a

ID:

n/a

Window title:

n/a

String 1:

Version of virus definitions on the system Scrip 250 ran on.

(e.g. 2.763.00)

String 2:

n/a

Text 1:

Virus definition version on the system where Scrip 250 ran, and the latest virus definition date on the sub-net.

(E.g.

Trend Micro virus definition versions on this system: 2.763.00, latest on network: 2.763.00

)

Text 2:

Version of virus definitions on the system Scrip 250 ran on.

(e.g. 2.763.00)

Text 3:

Date of latest virus definitions available on the sub-net (last modified date of virus definition files)

(E.g.

Sunday, August 7, 2005 16:39:32

)

Text 4:

Scrip execution type and anti virus product name and version

(E.g.

Data: Type of run: On-Demand
Data: Product: Trend Micro OfficeScan Client for Windows 2003/XP/2000/NT
Data: Version: 7.0

)
