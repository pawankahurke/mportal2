Scrip 24 - Silent.log Check

When an automated client update is performed, the installation is run in "silent" mode. If it encounters any errors or problems, it writes some diagnostic information in the file "silent.log" in the client directory. Scrip 24 looks for the "silent.log" file, and if it finds it, sends the content of this file as part of a log. This is very useful for diagnosing problems that arise from the installation in a client update.

The table below describes the information contained in the logs produced by this Scrip.


Client time: Date and time at which event was detected by the Client on the end-user system (e.g. 2001-03-02 19:21:36)
Server time: Date and time at which event log was received and stored in the SQL database on the Web log server (e.g. 2001-03-02 19:21:53)
Scrip number: Number of Scrip (no particular relevance)
Customer: Name of supported customer, individual department or site
Machine: Name of system as assigned at the time of Windows configuration
Username: Name of user when event occurred
Client version: Client version number (e.g. 0.700.2048.LE)
Client size: Size of Client in bytes
Priority: Priority of event ranging between 1 (lowest) and 9 (highest)
Description: Description of event that took place (e.g. symptom detected, problem resolved or automated procedure executed)
Type: n/a
Path: Location and name of orphaned log file on system that reported the event

(e.g. C:\PROGRAM FILES\...\CLIENT\temp\hfc1225.TMP)

Executable: n/a
Version: n/a
Size: n/a
ID: n/a
Window title: n/a
String 1: n/a
String 2: Client location on system that reported the event

(e.g. C:\PROGRAM FILES\...\CLIENT\CUST.EXE)

Text 1:

Text to introduce contents of silent.log file:

The following file contains error code for installshield.

Text 2: n/a
Text 3:

Contents of silent.log file

(e.g.

[InstallShield Silent]
Version=v5.00.000
File=Log File

[Application]
Name= Client
Version=0.700.4876.LE
Company=

[ResponseResult]
ResultCode=0

)

Text 4:

Location of silent.log file

(e.g.

Process: C:\PROGRAM FILES\...\CLIENT\silent.log

)
