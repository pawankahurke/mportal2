Scrip 60 - Clean Folders

Scrip 60 deletes files and directories specified in the Scrip's configuration page. All the configuration settings for Scrip 60 can be changed including file mask, locations, deletion of read-only files, and minimum age of files to be deleted. Its default settings are:

  • Locations scanned: Those matching the TEMP environment variable
  • Search criteria: [~*.*] [*.tmp] [$*.*]
  • Removal criteria: older than 14 Days, 0 Hours, 0 Minutes, 0 Seconds
  • Read-only files matching search criteria and aging criteria are not deleted
