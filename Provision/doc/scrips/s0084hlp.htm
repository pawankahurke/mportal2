Scrip 84 - Network Connectivity Status

Scrip 84 reports loss of TCP/IP network connectivity at the local interface, and can be configured to report when a network connection is re-established.

This Scrip generates a log both when network connectivity is first lost and when it is re-established.

When enabled, Scrip 84 will post an event log, and generate a message for the system's user if so configured, when the following thresholds are exceeded

  • No acknowledgment to three sends to another device on the sub-net within five seconds

  • No acknowledgment to three sends to the Internet within 20 seconds

The above thresholds are configuration parameters in Scrip 201 (Network Device Driver Management).
