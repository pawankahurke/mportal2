Scrip 222 - Report File Attributes

Scrip 222 reports the attributes of files and directories that match the mask specified in the Scrip's configuration page. All the configuration settings for Scrip 222 can be changed including file mask, and locations to be scanned. Scrip 222 reports the following attributes of a file:

  • File Name
  • File Version
  • File Size
  • File Creation Date
  • File Last Modified

Scrip 222 reports also folder properties. However, you should keep in mind that the only directory properties are Directory Name and Directory Creation Date.

The table below describes the information contained in the logs produced by this Scrip.


Client time: Date and time at which event was detected by the Client on the end-user system (e.g. 2001-03-02 19:21:36)
Server time: Date and time at which event log was received and stored in the SQL database on the Web log server (e.g. 2001-03-02 19:21:53)
Scrip number: Number of Scrip (no particular relevance)
Customer: Name of supported customer, individual department or site
Machine: Name of system as assigned at the time of Windows configuration
Username: Name of user when event occurred
Client version: Client version number (e.g. 0.700.2048.LE)
Client size: Size of Client in bytes
Priority: Priority of event ranging between 1 (lowest) and 9 (highest)
Description: Description of event that took place (e.g. symptom detected, problem resolved or automated procedure executed)
Type: n/a
Path: n/a
Executable: n/a
Version: n/a
Size: n/a
ID: n/a
Window title: n/a
String 1: Creation date and time of the last file deleted by the Scrip

(e.g. Friday, December 22, 2000 10:58:24)

String 2: Client location on system that reported the event

(e.g. C:\PROGRAM FILES\...\CLIENT\CUST.EXE)

Text 1: Total number of files and directories whose properties were reported

(E.g.

Attributes for 2 files and 0 directories were reported

)

Text 2:

Scrip 222 configuration parameters (location scanned for files, search criteria, type of run), and results of search operations

)
