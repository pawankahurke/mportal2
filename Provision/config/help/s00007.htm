Scrip 7 Configuration

Scrip 7 detects whether any of the periodic e-mail logs (produced by Scrips 5, 66 and 67) have been deleted before they could be sent via e-mail, and if so, reports that fact.

Scrip 7 Default Configuration Parameter

  • Scrip enabled

Even if enabled, Scrip 7 does not have any effect if web logging is in use, which is the default.
