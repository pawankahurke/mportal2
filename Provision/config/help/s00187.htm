Scrip 187 Configuration

Configuration parameters

  • Announce presence

  • Report current machine list

Notes

In order to have Scrip 187 log the most up-to-date list of systems where the ASI client is installed and running on the same sub-net as the system where Scrip 187 was executed, you should:

  • Click on the the Announce presence Execute button. Wait for about 30 seconds after the command has been executed, then

  • Click on the the Report current machine list Execute button

Announce presence

Clicking on the Execute button in the Announce presence configuration parameter will cause the execution of a singled UDP broadcast targeting port 2721 from the system where Scrip 187 was executed.

Report current machine list

Clicking on the Execute button in the Report current machine list configuration parameter will cause Scrip 187 to log to the ASI server the Local Machines page accessed via http://localhost:2721, on the system where Scrip 187 was executed.
