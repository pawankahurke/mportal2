Scrip 45 Configuration

Default Configuration Parameters

  • Scrip enabled
  • Ports to include for logging set to include all ports
  • Ports to exclude for logging set to exclude none

Ports to include for logging

The Ports to include for logging list contains the IP services ports that you want Scrip 45 to monitor for probing by another system.

As indicated by the *, by default this list includes all IP services ports.

Entries in the Ports to include for logging list should be separated by a comma with no spaces.

Ports to exclude for logging

The Ports to exclude for logging list contains the IP services ports that you want Scrip 45 to not monitor for attempts to probe by other systems. You might want to use this parameter to list ports that you know are being scanned for a known reason. In such a case Scrip 45's detection of a port probe would be superfluous.

By default, this list is empty.
