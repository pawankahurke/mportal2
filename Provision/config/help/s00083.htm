Scrip 83 Configuration

Default Configuration Parameters

  • Scrip enabled
  • User interface enabled disabled
  • Seconds between retries 60 seconds

User Interface

Enabling this parameter lets end-users receive messages from the Scrip informing them about the failure that took place, the status of the failed service and other support-related information.

Seconds between retries

Users can choose the time period between attempts by Scrip 83 to check if the failed service is restored. This time period is measured in seconds.
