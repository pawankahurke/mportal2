Scrip 245 Configuration

Configuration parameters and their default values

  • Close feedback page? is enabled

  • Name: (Blank defaults to system user) is empty

  • Company: (Blank defaults to site name) is empty

  • Start session URL is empty

  • Name of agent or representative: is empty

  • Run now

  • Shutdown chatlink

  • Send Ctrl-Alt-Del

Notes

Please note that you can configure Scrip 245 in two ways:

  • Accessing its configuration page either on your ASI server, or directly at a site, or

  • Using the remote control configuration wizard on the ASI server (https://asiserverdomain/main/config/remote.php)

Before using Scrip 245, you should increase the frequency of execution of Scrip 177 (Scrip Configuration Update) to ensure that execution of Scrip 245 will occur quickly after you trigger its execution from the ASI server. We suggest that Scrip 177 be configured to run once every three minutes. While it's not necessary, you may want to revert the Scrip 177 schedule to the settings you normally use after the remote control sessions are over.

Close feedback page? (optional)

At the end of a session, Goto Assist, opens a browser window and loads a user response page. When enabled, the Close feedback page configuration parameter causes Scrip 245 to automatically close the user response browser window. By default, Close feedback page is enabled.

Name: (Blank defaults to system user) (optional)

The Name: (Blank defaults to system user) configuration parameter contains the name of the user you are having a remote support session with. Please note that this configuration parameter is optional. Typically, leaving this configuration parameter blank should work as when this is the case Scrip 245 automatically uses the name of the user currently logged in.

Company: (Blank defaults to site name) (optional)

The Company: (Blank defaults to site name) configuration parameter contains the name of the user you are having a remote support session with. Please note that this configuration parameter is optional. Typically, leaving this configuration parameter blank should work as when this is the case Scrip 245 automatically uses the name of the site the system where Scrip 245 runs belongs to.

Start session URL (mandatory)

You need to fill in the Start session URL configuration parameter as it is required by Goto Assist. Scrip 245 will not work unless you enter the correct information in this configuration parameter.

Start session URL is the URL of the page users are directed to in order to start a GTA session. Citrix supplies that information to licensees.

Name of agent or representative: (mandatory)

You need to fill in the Name of agent or representative: configuration parameter as it is required by Goto Assist. This configuration parameter contains the identifier of the support engineer managing the remote support session. For example, if your organization has support engineers identified as:

  • Help Desk1

  • Help Desk2

  • Help Desk3

In the Name of agent or representative: configuration parameter , you would need to enter the identifier of the support engineer managing the remote support session.

Please note that Scrip 245 will not work unless you enter the correct information in this configuration parameter.

Run now

Clicking on the Execute button in the Run now configuration parameter will cause Scrip 245 to open a remote support session using the information entered in the Scrip's configuration parameters.

Shutdown chatlink

Clicking on the Execute button in the Shutdown chatlink configuration parameter will cause Scrip 245 to shut down the chat link used by GoToAssist.

Send Ctrl-Alt-Del

Clicking on the Execute button in the Send Ctrl-Alt-Del parameter will cause Scrip 245 to execute a ctrl+alt+del command on the system where it is running, bringing up on the system's screen the login dialogue box. In this way, you can use GoToAssist on unattended systems. This is not possible with GoToAssist on its own.
