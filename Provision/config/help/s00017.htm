Scrip 17 Configuration

Default Configuration Parameters

  • Scrip disabled
  • Executables to detect list empty
  • Command Line arguments to detect list empty

Executables

Scrip 17 allows the detection of multiple executables entered in the Executables to detect window.

You should enter the full name of the executable, including extension. Location path should not be entered.

When more than one executable is entered, each name should be on a separate line.

Scrip 17 detects the execution of the program(s) entered in the Executables to detect list by matching these names with those of processes that are active on the system.

Scrip 17 detects all types of executables except for those with the .com and .bat extensions.

Command Line Arguments

Scrip 17 allows the detection of arguments entered by the user in the Command Line arguments to detect window.

You should enter the argument you want detected exactly as it is used in the command line of the executable.

If you want Scrip 17 to detect more than one argument, each argument should be entered on a separate line.
