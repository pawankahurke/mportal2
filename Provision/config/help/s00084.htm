Scrip 84 Configuration

Configuration parameters and their default values

  • Scrip enabled disabled

  • User interface enabled disabled

  • Dialog box timeout (sec) if no user present set to 30 seconds



Scrip 84 tests the availability of TCP/IP network connectivity at the local interface.

When enabled, Scrip 84 will post an event log, and generate a message for the system's user if so configured, when the following thresholds are exceeded

  • No acknowledgment to three sends to another device on the sub-net within five seconds

  • No acknowledgment to three sends to the Internet within 20 seconds

The above thresholds are configuration parameters in Scrip 201 (Network Device Driver Management).

User interface enabled

The user interface for Scrip 84 consists of two dialog boxes. The first one lets end-users know that they lost network connectivity. The second one informs them when network connectivity is re-established.

Dialog box timeout (sec) if no user present

In the Dialog box timeout (sec) if no user present configuration parameter you enter the amount of time that Scrip 84 will wait before acting on behalf of the user to either close a dialog box or perform the default action presented by a dialog box. The default value is 30 seconds.
