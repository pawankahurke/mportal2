Scrip 42 Configuration

Default Configuration Parameters

  • Scrip enabled
  • Timeout for Ctl-Alt-Del (seconds) 60
  • Dialog text You have been locked out of your system because of a wrong password. Please click on OK, then press the Ctrl, Alt, and Del keys simultaneously to automatically reset your password.

Timeout for Ctl-Alt-Del (seconds)

The parameter Timeout for Ctl-Alt-Del (seconds) lets you define the amount of time the Scrip will wait for the user to type Ctl-Alt-Del after he/she has clicked OK in the dialog box containing the Dialog text. If the user waits longer than that, Scrip 42 just completes execution and reports that the password was not reset.

Dialog text

This parameter is used to enter the content of the message displayed when a user is locked out of his/her system because of having entered an incorrect password for a number of times that exceeeds the number of failures allowed as defined in the system's security policy.

Important note

Please note that in the default configuration on Microsoft Windows 2000 and XP based systems, the need to press simultaneously the Ctrl-Alt-Del keys at start-up before logging onto a system is disabled. In this case, Scrip 42 automatically initiates the Courier Direct password reset procedure without displaying the dialog box with the message contained in Dialog text. In order to account for this system configuration, Scrip 42 delays taking action for 1-2 seconds, to see if the Courier Direct dialog comes up, before displaying the dialog box with the Dialog text message.

We should also note that on Microsoft Windows XP based systems Scrip 42 will not work when the "Welcome Screen" feature is enabled This is also true for the Courier Direct application. In fact, it disables the "Welcome Screen" feature.
