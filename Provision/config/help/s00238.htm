  • When enabled, Scrip 238 runs on Fridays at 3 AM

  • Log Symantec virus definition dates now

Report when anti-virus product is not installed

If you enable the Report when anti-virus product is not installed configuration parameter, on systems where Norton Anti Virus is not installed Scrip 238 will post a log on the ASI server with this information.

You might want to use this configuration parameter periodically to find out which systems at a site do not have anti virus software installed. In this way, you would be able to intervene pro-actively and add anti virus protection to unprotected systems before a virus infects them.

Log Symantec virus definition dates now

With the Execute button, you can run Scrip 238 on demand, even if it's not enabled. In Scrip 238, the Execute button can be used to execute the Scrip on the local system or on all supported systems.
