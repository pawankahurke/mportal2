Scrip 11 Configuration

Scrip 11 detects whether any pending e-mail logs that should be in the "temp" directory of the Client folder are missing, and if so, reports that fact.

Scrip 11 Default Configuration Parameter

  • Scrip enabled

Even if enabled, Scrip 11 does not have any effect if web logging is in use, which is the default.
