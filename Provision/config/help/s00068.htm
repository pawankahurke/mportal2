Scrip 68 Configuration

Default Configuration Parameters

  • Scrip enabled
  • User interface enabled enabled

Scrip 68 detects date/time changes of more than five hours. Any date/time change of five hours or less will be ignored.

User Interface enabled

Enabling this parameter lets the Automated Support Infrastructure Client tell the end-user that he/she (inadvertently) changed the system date/time, and offer the option to reset it.

If the user interface is not enabled, Scrip 68 will detect the date/time change but will not take any action to reset it.
