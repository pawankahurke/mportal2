Scrip 71 Configuration

Default Configuration Parameters

  • Scrip enabled
  • Uninstall executables to detect The Client triggers the execution of Scrip 71 when it detects the execution of a software removal program that matches the following files masks:
    • isuninst.exe
    • uninst*.exe
    • unwise*.exe
    • msiexec.exe

Uninstall executables to detect

You can add file masks to the Uninstall executables to detect parameter, one per line, at the beginning, end or anywhere in the middle of the default list.
